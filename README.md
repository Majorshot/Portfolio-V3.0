# Portfolio V3.0 — The Digital Arsenal

[![License: MIT](https://img.shields.io/badge/License-MIT-orange.svg)](https://opensource.org/licenses/MIT)
[![Tech Stack](https://img.shields.io/badge/Stack-MERN%20%2F%20GSAP%20%2F%20Tailwind-black)](https://github.com/Majorshot/Portfolio-V3.0)

A high-performance, Awwwards-inspired portfolio designed through a brutalist lens. This project isn't just a website; it's a visceral digital identity built to showcase premium engineering, sophisticated motion design, and multi-disciplinary expertise.

> **"I don't just write code; I architect entire digital identities."**

---

## ⚡ Core Engine (Tech Stack)

*   **Frontend:** [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Dark-first implementation)
*   **Motion & Physics:** [GSAP](https://greensock.com/gsap/) + [Framer Motion](https://www.framer.com/motion/)
*   **Assets:** [Opus](https://en.wikipedia.org/wiki/Opus_(audio_format)) High-Efficiency Audio + Optimized WebM Video
*   **Form Logic:** [Web3Forms](https://web3forms.com/) integration with multi-step validation

---

## 🎨 Aesthetic Highlights

### 🕹️ Brutalist HUD Overlay
A persistent system-inspired HUD that tracks system time, memory allocation (hex), and scroll position in real-time, creating a terminal-like immersive environment.

### 🌊 Grainient Architecture
Dynamic, scroll-reactive animated gradients with high-fidelity noise/grain filters, providing a textured, premium backdrop for content sections.

### 🔊 Audio-Reactive Experience
A custom-built media player featuring high-efficiency `.opus` streams (Toxic, DNA, Stargazing). The UI reacts to the sonic landscape, creating a cohesive audiovisual journey.

### 🖼️ Parallax Project Showcase
A high-performance scroll experience where project visuals float and transition through direct DOM manipulation and GPU-accelerated transforms.

---

## 🚀 Key Features

*   **Premium Loading Sequence:** A terminal-style cyberpunk entry transition.
*   **Photography Archive:** A high-performance horizontal filmstrip scroll optimized for 60fps interaction.
*   **Multi-Step Contact Protocol:** A sophisticated, state-managed form with real-time field validation.
*   **Staggered Typography:** Blur-reveal and text-pressure animations that respond to scroll depth.
*   **Full Theme Control:** Seamless transition between Brutalist Dark and Minimal Light modes.

## 🛠️ Installation & Setup

1.  **Clone the Arsenal:**
    ```bash
    git clone https://github.com/Majorshot/Portfolio-V3.0.git
    ```

2.  **Initialize Dependencies:**
    ```bash
    npm install
    ```

3.  **Ignite Development Server:**
    ```bash
    npm run dev
    ```

4.  **Production Build:**
    ```bash
    npm run build
    ```
